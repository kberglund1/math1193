\maketitle

Solve the following polynomial equations.

\begin{enumerate}
    \item \((x - 1)(x + 2) = 0\)
    \item \((x - 1)^2(x + 2) = 0\)
    \item \(-3(x - 1)^2(x + 2)\)
    \item \((2x - 1)^3(x + 1)\)
    \item \(2x^3 + 8x^2 + 3x + 12 = 0\)
    \item \(2x^2 + 7x + 3 = 0\)
\end{enumerate}

How could you check your answers? 

Which equations were most difficult to solve?
