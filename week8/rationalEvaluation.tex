\title{Rational Evaluation and Equations}
\author{Kenneth Berglund}

\begin{document}
\begin{abstract}
    Evaluating rational function and solving rational equations.
\end{abstract}
\maketitle

For each of the following functions, solve the equation \(f(x) = 0\)
and evaluate \(f(0)\). What do these answers tell you about the graph 
of \(f\)?

\begin{enumerate}
    \item \(f(x) = \frac{x^2 - 1}{x + 2}\)
    \item \(f(x) = \frac{x + 1}{x^2 - 1}\)
    \item \(f(x) = \frac{1}{x - 3} + \frac{2}{x + 1}\)
    \item \(f(x) = \frac{2x + 1}{x^ - x}\)
\end{enumerate}
